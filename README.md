# SmartCart-Segmentation-System

This project uses unsupervised machine learning to segment e-commerce customers into meaningful groups based on demographics, spending behaviour, purchase frequency, and engagement patterns.

## Tech Stack
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-Learn

## Workflow
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature transformation for clustering
- Customer segmentation using clustering techniques
- Analysis of customer groups based on spending and purchase behaviour

## Goal
To identify different customer segments for targeted marketing, personalized engagement, and customer retention strategies.
